# Data Redundancy Removal System

## Internship
This project is developed as part of the CodeAlpha Python Internship.

## Description
A Python-based system that validates data and prevents duplicate entries
from being stored using hash comparison.

## Technologies Used
- Python
- JSON
- SHA-256 Hashing

## How It Works
1. Validates incoming data
2. Generates unique hash
3. Checks redundancy
4. Stores only unique records
